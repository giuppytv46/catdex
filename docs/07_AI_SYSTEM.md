# CatDex AI System Specification v1.0

## Purpose

The AI system is the core mechanic of CatDex.

Its role is NOT simply identifying a cat.

Its role is transforming a real-world encounter into a collectible game experience.

The AI should always maximize player satisfaction while remaining honest about uncertainty.

# Processing Time

<5 seconds

# Cost Optimization

Resize image before upload.

Do not analyze duplicates multiple times.

Cache previous analyses.

Use background processing.

---

# Safety

Never identify people.

Ignore humans.

Focus only on cats.

Reject inappropriate images.
