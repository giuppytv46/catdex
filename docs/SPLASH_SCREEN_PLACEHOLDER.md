# CatDex Splash Screen Placeholder

CatDex currently uses native placeholder splash resources.

## Android

- `android/app/src/main/res/drawable/launch_background.xml`
- `android/app/src/main/res/values/styles.xml`

## iOS

- `ios/Runner/Base.lproj/LaunchScreen.storyboard`

## Future Requirements

Before store submission, replace placeholders with a polished launch experience that:

- Matches the CatDex light and dark theme.
- Loads quickly.
- Avoids heavy animation in the native splash.
- Does not show network-dependent content.
- Transitions smoothly into the Flutter splash/onboarding flow.

## Visual Direction

Use the CatDex design system:

- Bright green, purple, and blue palette.
- Simple centered mark.
- Friendly cartoon mood.
- Clean background that works on small devices.
