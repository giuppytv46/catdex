# CatDex App Icon Placeholder

CatDex currently uses placeholder launcher icon resources.

## Current Android Placeholder

- `android/app/src/main/res/drawable/ic_launcher.xml`
- Referenced by `android/app/src/main/AndroidManifest.xml`

## Current iOS Placeholder

The iOS project does not yet include a final production app icon asset catalog.

## Future Requirements

Before store submission, create production icon assets for:

- Android adaptive icon foreground/background.
- Android legacy launcher icon fallback.
- iOS AppIcon asset catalog at all required sizes.

## Visual Direction

The production icon should match the CatDex design system:

- Friendly cat silhouette or collectible-card motif.
- Bright green, purple, and blue palette.
- Rounded, soft, playful shape language.
- Legible at small sizes.

## Rules

- Do not use copyrighted characters.
- Do not use real brand logos.
- Do not include store badges.
- Keep source artwork in a tracked design handoff folder only if licensing is clear.
