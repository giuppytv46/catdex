# CatDex Gameplay Design v1.0

## Purpose

This document defines every gameplay loop of CatDex.

The objective is to create a relaxing, rewarding and highly replayable collecting game.

The player should always feel excited about discovering "one more cat".

---

# Core Philosophy

CatDex is NOT a simulator.

CatDex is NOT a cat encyclopedia.

CatDex is a collection game.

Everything must reinforce three emotions:

* Curiosity
* Discovery
* Collection

---

# Anti-Frustration Rules

Never punish players.

No energy system.

No pay-to-win.

No impossible timers.

Duplicates are always useful.

Progress is always visible.

---

# Success Metric

A successful session should leave the player thinking:

"I'll go for one more cat tomorrow."

This feeling is the primary design goal of CatDex.
